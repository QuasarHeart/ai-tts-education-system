# ai-tts-education-system
AI-powered teaching voice processing system based on speech synthesis (TTS)

基于 AI 语音合成（Text-to-Speech, TTS）的教学声音处理系统，面向教育场景提供自然、清晰、可扩展的语音讲解与有声课件生成能力。

## 项目背景

随着人工智能技术的发展，语音合成（TTS）技术在智能助手、有声阅读、自动播报等领域已得到广泛应用。在教育场景中，TTS 技术能够降低教师重复性工作成本，提升教学内容的可访问性，并在师资不足或特殊教育场景中发挥重要作用。

本项目围绕“教学声音处理”这一核心需求，设计并实现一个基于 AI 语音合成的教学软件系统，支持多种教学内容的语音生成与处理形式，探索 TTS 技术在智能教育领域的应用潜力。

## 功能概述

- **声音样本库管理**：支持预设标准声音，以及通过上传或录制方式构建个性化声音样本库  
- **个性化语音讲解**：输入教学文本，选择指定声音生成教学语音  
- **标准语言输出**：支持普通话、英文等标准发音，可调节语速、语气与节奏  
- **有声课件制作**：上传教学课件（如 PPT），自动生成可下载的有声课件  
- **扩展功能（可选）**：声音置换、字幕生成与音视频同步处理  

## 系统架构概览

系统采用前后端分离与服务化架构，主要包括：

- 前端展示层：负责用户交互与教学内容配置  
- 后端业务层：负责任务调度、文件管理与接口服务  
- 语音合成服务（TTS Service）：基于预训练模型完成文本到语音的生成  
- 存储与资源管理：管理音频文件、课件与中间结果  

## 技术选型（初步）

- 前端：Web 前端框架（如 Vue / React）  
- 后端：Python + Web 框架（FastAPI / Flask）  
- 语音合成：基于开源预训练 TTS 框架（如 Coqui TTS）  
- 音频处理：FFmpeg / Python Audio Libraries
